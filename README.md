# Node Server Boilerplate #
This repository is a node.js server boilerplate project using Sequelize ORM

### Required modules ###
* grunt (task runner): npm install grunt-cli -g
* jshint (linter): 

### Debugging ###
to run this project locally, navigate to the project folder in the terminal and execute "DEBUG=bradslist-web:* ./bin/www"
